# ClockWise - Learn to Tell Time

An interactive clock reading game designed for children with learning difficulties. Built with React + Vite.

## Features

- 12 progressive levels from basic hour reading to 24-hour time
- Multiple input methods (text, visual selection, multiple choice)
- Adaptive difficulty based on performance
- FNAF-themed characters and rewards
- Robux counter for engagement
- Progress tracking and analytics
- Accessibility options (fonts, sizes, high contrast)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `src/components/` - React components
- `src/hooks/` - Custom React hooks
- `src/utils/` - Utility functions (storage, levels, audio, etc.)
- `public/images/` - Static assets
